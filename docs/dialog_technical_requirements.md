# Формат записи реплик
[speaker_index, string, num], где:
- `speaker_index` - номер говорящего
- `string` - текст реплики
- `num` - ширина окна диалога (в пикселях)
- При `num == 0` используется стандартный размер окна
# Поведение окон
Каждый спрайт `spr_dialog_box_x` вместе с `x_spr_dialog_anim` появляется в заданном месте для каждого говорящего, также можно указать в качестве место дефолт, который будет работать для модельки игрока `obj_yura`
